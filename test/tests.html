🧪 StadiumIQ — Core Logic Test Suite

Zero-dependency, browser-runnable tests for the pure/critical functions in index.html. Open this file directly in a browser — no build step, no npm install.

📋 Manual test checklist (not automatable without a real API key / live browser demo) 

[ ] Chat: with no API key set, every persona gets a relevant fallback response (not blank/broken)
[ ] Chat: with a valid API key, switching persona changes the assistant's tone/vocabulary noticeably
[ ] Dashboard: "AI Operational Summary" shows a loading spinner then real content (not the old static text)
[ ] Dashboard: with no API key, the summary panel clearly shows "Demo mode" rather than pretending to be live
[ ] Crowd page: "Analyze with AI" button works and recovers gracefully if Gemini returns malformed JSON
[ ] Journey Planner: entering a custom "from" location produces a route that actually references that location
[ ] Journey Planner: rapid repeat clicks don't exceed the 10-calls/minute rate limit (check console/toast)
[ ] Settings modal: Tab key stays trapped inside the modal while open; Escape closes it
[ ] Settings modal: focus returns to the element that opened it after closing
[ ] Chat message list: a screen reader (VoiceOver/NVDA) announces new AI messages as they arrive
[ ] Gate list: each gate item is reachable and activatable via keyboard only (Tab + Enter/Space)
[ ] Language switch: selecting a previously-"missing" language (de/zh/ja/hi/ko) no longer silently falls back to English in the AI chat
